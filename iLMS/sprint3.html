Sprint 3

i-LMS project

Sprint Plan

In this sprint, we worked on the integration part which was incomplete in the previous sprint. We also worked on a new user-story which was added to Sprint backlog.

User Stories

In this sprint, one user-story was carried forward from the previous sprint and one new user-story was added to the backlog. These were:

  1. As a student, I should be able to start the assessment to see the first question, so that I can read the question and answer it. (Continued from sprint 2)
  2. As a student, I should be able to see the result of my assessment after I finish, so that I can move on to the next learning objective.

My Role

Since I was working with the code editor part in the previous sprint, in this sprint I was asked to change the supporting library to the latest version of codemirror, i.e., CodeMirror 6. Using the latest version proved to be better as this library already had a basic setup so I didn't had to select which each and every feature which I want in my editor. Moreover, I was able to style the editor according to my wishes rather than using a fixed theme.

I also had the responsiblity to integrate the editor with the questions page and style it. I also had to make sure that the design of the editor was responsive.

Sprint Outcome

In this sprint, we were able to complete all the user-stories as planned. However, few bugs were identified whose resolution will be done in the next sprint.
